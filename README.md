# Sunrise-Lasertag
A self-sufficient, portable lasertag system with customizable weapons. Made for you and your friends to take and play wherever and whenever you want.

## Usage
This is the main repository of this project, containing three submodules. You should not clone this repository, but rather the individual submodules. 

To use or edit this codebase, clone the **Library** repository into your Arduino libraries directory and the **Devices** repository into your Arduino projects directory. You'll also need the **IRremoteESP8266** library and the ESP32 boards installed in your Arduino IDE. [Read this documentation for help!](https://docs.espressif.com/projects/arduino-esp32/en/latest/installing.html) 

The **Server** repository is to be cloned onto a Raspberry Pi or any other device you want to use as a server. This device must be able to act as a network hotspot.
